# 样式规则

## 颜色规范

- **背景颜色**: #1a1a1a
- **主题颜色**: #2ecc71
- **普通文本颜色**: #e5eaf3
- **高亮文字**: 使用主题色 #2ecc71

## Element UI 主题配置

- **主题色**: #2ecc71
- **背景颜色**: #1a1a1a（替换默认白色背景）
- **配置文件**: `src/styles/element-variables.scss`
- **引入方式**: 在 `main.ts` 中引入自定义主题

## 全局样式

- 使用 CSS 变量定义颜色
- 统一设置 body 样式
- 规范链接和按钮样式

## 组件样式

- **命名规范**: 使用 BEM 命名规范
- **文件命名**: 组件样式文件与组件文件同名
- **语法规范**: 样式使用 SCSS 嵌套语法

## 响应式设计

- **断点设置**:
  - 移动端: < 768px
  - 平板: 768px - 1024px
  - 桌面: > 1024px

## 测试与验证

- 检查所有页面的背景颜色是否为 #1a1a1a
- 检查主题色 #2ecc71 是否在所有组件中正确显示
- 检查文本颜色 #e5eaf3 是否在所有文本中正确显示
- 验证 Element UI 组件的主题颜色是否已更新为 #2ecc71
- 测试响应式设计在不同设备尺寸下的表现
